### Social Sharer

> The final social share module ***made in plain javascript***

## Demo

## Features

- Plain js module, you can use it with whaterever framework
- No dependencies
- Themes support

## Install

```
npm install socialsharer
```

## Usage

It's so simple

```js
Share(
  document.querySelector('#social'),
  {
    url: 'http://www.nytimes.com/',
    networks: ['tw', 'fb', 'in']
  }
);
```
